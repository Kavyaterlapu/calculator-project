# 🧮 Calculator Web App

A simple calculator built with HTML, CSS, and JavaScript.

## ✨ Features
- 🧠 Basic math operations
- 🔄 Clear and All Clear buttons
- 🧱 Clean UI with CSS Grid
- ⚠️ Error handling

## 📂 Files
- `calculator.html`
- `calculator.css`
- `calculator.js`

## 🚀 How to Use
1. Clone the repo
2. Open `calculator.html` in a browser
3. Start calculating! ➕ ➖ ✖️ ➗
